#User Stories
Возможный функционал:
1. Часть приложения - сервис, работает в фоновом режиме.
2. Создает уведомления каждый час (опционально) с информацией о погоде.
3. В настройках задаются: 
	- город, 
	- таймер уведомлений, 
	- вкл/откл уведомления, 
	- текущий шлюз API (можно предусмотреть несколько мест, откуда брать данные). 
4. Возможность оповещения/создания постов в соц.сетях.
5. При открытии активити приложения, можно выбрать нужный город и посмотреть погоду на текущий момент/день/пару дней/неделю.
6. Отображение погоды на карте.
